$ npm install --global vue-cli
$ vue init webpack my-project

$ cd my-project
$ npm install
$ npm run dev

　　在用nodejs安装vue项目时，不选择自动执行npm install，而在之后自己进入项目目录手动用cnpm install，为了安装node_modules目录。

　　运行$ npm run dev，之后可以进入本地服务器指定端口查看实时编译的效果。
　　退出监听Ctrl+C。
　　运行$ npm run build，编译生成文件，之后可以将生成的文件放入项目框架中。
　　或在phostorm打开项目后(必须将此项目设置为唯一打开的根目录)，
Run->Edit Configurations->左上角的+号->npm->在Scripts的下拉菜单中找到dev和build，并分别以此为Name新建。点击绿色三角形即运行dev就是开启实时效果，运行build就是编译生成结果(项目目录下会出现一个dist目录，此目录内即编译结果)。

　　用phpstorm打开项目中的index.js后出现大量的红线勾画，这是默认的js版本太低，进入Settings->Languages & Frameworks->选中JavaScript->JavaScript Language version选择“ECMAScript 6”。
　　用phpstorm打开.vue后没有html字体色，Settings -> plugins -> 搜索vue.js -> search in repositories ->选择Vue.js并Install。之后重启phpstorm。

　　安装jquery：
$ cnpm install jquery --save
（--save是把jquery的配置保存到根目录下的package.json文件，包括版本号，下次install则可以根据版本号进行安装）
　　在要使用jquery的文件内使用 import $ from 'jquery'
