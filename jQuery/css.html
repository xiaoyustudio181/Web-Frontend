• .css() 获取或设置样式。
    $('#app1').css('color')
    $('#app1').css('border')
    $('#app1').css('color','blue')
    $('#app1').css('border','3px solid pink')
    $('#app1').css({fontSize:'25px',border:'4px dotted green'})
    This is an area.
  • .width(), .height() 获取或设置宽高。（不包括内边距、边框、外边距）
    $('#app2').width()
    $('#app2').height()
    $('#app2').width('200px')
    $('#app2').height('25px')
  • .innerWidth(), .innerHeight() 获取或设置宽高。（包括内边距）
    $('#app3').innerWidth()
    $('#app3').innerHeight()
    $('#app3').innerWidth('200px')
    $('#app3').width()
    $('#app3').innerHeight('50px')
    $('#app3').height()
  • .outerWidth(), .outerHeight() 获取或设置宽高。（包括内边距、边框，加上参数true则包括外边距）
    $('#app4').outerWidth()
    $('#app4').outerHeight()
    $('#app4').outerWidth('200px')
    $('#app4').width()
    $('#app4').outerHeight('60px')
    $('#app4').height()
    $('#app4').outerWidth(true)
    $('#app4').outerHeight(true)
    $('#app4').outerWidth('200px',true)
    $('#app4').width()
    $('#app4').outerHeight('80px',true)
    $('#app4').height()
  • .offset() 获取相对于 document 的坐标。
    $('#app5').offset()
  • .position() 获得相对于最近的有定位的祖先元素的坐标。
    $('#app6').position()
