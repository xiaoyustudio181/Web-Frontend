  4. 自定义方法：取随机数。
    function rand(min, max) {
    return Math.floor(Math.random() * (max - min + 1) + min);
}
    rand(10,16)
  5. 去除一个数的小数部分，返回整数部分。Math.trunc

    Math.trunc(-4.1) // -4
    Math.trunc(-4.9) // -4
    Math.trunc(-0.1234) // -0
    Math.trunc('123.456') // 123
