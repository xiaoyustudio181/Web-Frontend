  4. localStorage 生命周期是永久，除非用户清除信息。
    localStorage
    localStorage['test_value'] = 100
    localStorage['test_value2'] = 200
    localStorage['test_value']
    localStorage.getItem('test_value')
    localStorage.removeItem("test_value")
    delete localStorage['test_value']
    localStorage.clear()
    for (var i = 0; i < localStorage.length; i++) {
    var name = localStorage.key(i);
    var value = localStorage.getItem(name);
    console.log(name+': '+value);
}
  5. sessionStorage 生命周期为当前窗口或标签页。
